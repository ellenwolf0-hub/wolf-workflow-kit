# Demo Scripts

This folder contains materials for live demos of the Wolf Workflow Kit. **It does not ship to users.**

| File | Purpose |
|------|---------|
| `demo-script.md` | Step-by-step walkthrough for a live demo |
| `orient-vault-setup.md` | Pre-demo vault setup — creates fake-but-realistic files so `/orient` surfaces a convincing open pickup |
| `transcript-calibration-taylor-chen.txt` | Fictional Granola transcript used in the demo to show `/meet` in action |
| `pitch.md` | Talking points for introducing the kit to a new potential user |

## Important

Taylor Chen, Jordan Kim, and Marcus Rivera are **fictional names** created for demo realism. They are not real Grammarly employees.

The `orient-vault-setup.md` script creates temporary files in the vault for the demo. After the demo, delete:
- `vault/Daily/DN - 2026-04-08.md`
- `vault/Pickups/PIC - Taylor Chen Comp Analysis.md`

These files are intentionally excluded from the vault template so new users start with a clean install.
